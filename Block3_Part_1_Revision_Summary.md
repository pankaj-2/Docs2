## Chapter 7: Production Function
**Overview:** A production function shows the maximum output a firm can get from a given set of inputs. This chapter covers how managers decide *what input combination* to use and *what technology* to use.

---

### 7.1 What is a Production Function?
- **Definition:** Q = f(x₁, x₂, …, xₙ) — relates inputs to maximum possible output
- With two inputs (capital K, labour L): **Q = f(L, K)**
- Output can be tangible (TVs) or intangible (healthcare)

---

### 7.2 Technical vs Economic Efficiency

```
INPUT COMBINATIONS GIVEN ONLY
        ↓
  Technical Efficiency
  (Max output from given inputs)

INPUT COMBINATIONS + INPUT PRICES GIVEN
        ↓
  Economic Efficiency
  (Lowest cost for given output)
```

- A **technically efficient** firm gets maximum output from any input combo
- An **economically efficient** firm produces a given output at the lowest possible cost
- A technically efficient firm is NOT necessarily economically efficient

---

### 7.3 Short Run vs Long Run

```
SHORT RUN → At least ONE input is FIXED (e.g., plant/machinery)
LONG RUN  → ALL inputs are VARIABLE (firm can change everything)
```

- **Fixed input:** Cannot be changed quickly (e.g., factory building)
- **Variable input:** Can be changed quickly (e.g., number of workers)

---

### 7.4 Production with ONE Variable Input (TP, AP, MP)

- **Total Product (TP):** Total output at each level of variable input
- **Average Product (AP):** AP = Q/L (output per worker)
- **Marginal Product (MP):** MP = ΔQ/ΔL (extra output from one more unit of input)

#### Mini Example (from Table 7.1):
| Workers (L) | TP | MP | AP |
|---|---|---|---|
| 3 | 54 | 26 | 18 |
| 4 | 76 | **22** | 19 |
| 5 | 90 | 14 | 18 |

> MP₄ = 76 − 54 = **22** (output added by 4th worker)

---

### 7.5 Relationships Between TP, MP, AP

```
MP > 0  →  TP is RISING
MP = 0  →  TP is at MAXIMUM
MP < 0  →  TP is FALLING

MP > AP  →  AP is RISING
MP = AP  →  AP is at MAXIMUM (intersection point)
MP < AP  →  AP is FALLING
```

---

### 7.6 Law of Diminishing Marginal Returns
- As you keep adding one more unit of a variable input (other inputs fixed), MP eventually **starts to fall**
- Key conditions: technology is fixed; at least one input is fixed
- Example: Adding fertilizer — 1st dose +50%, 2nd +30%, 3rd +20% … eventually output drops

---

### 7.7 Three Stages of Production

```
STAGE I   → MP > 0, AP rising  (MP > AP)  — NOT rational to stop here
STAGE II  → MP > 0, AP falling (MP < AP)  — OPTIMAL range ✓
STAGE III → MP < 0, TP falling             — NOT rational to produce here
```

- **Rational producers always operate in Stage II**

---

### 7.8 Profit Maximisation Rule (One Variable Input)
- Keep hiring until: **Value of MP = Price of input**
- i.e., MRP of Labour > Price of Labour → keep hiring
- Stop when: MRP = P_L

---

### 7.9 Production with TWO Variable Inputs — Isoquants

- **Isoquant:** Curve showing all combos of K and L that produce the **same output**
- Higher isoquant = higher output
- Isoquants **cannot intersect**
- A set of isoquants = **Isoquant Map**

```
         K
         |      Q₃ = 20
         |   Q₂ = 15
         | Q₁ = 10
         |_______________L
```

---

### 7.10 Marginal Rate of Technical Substitution (MRTS)

- **MRTS** = how much K you give up for 1 more unit of L, keeping output constant
- **MRTS = |ΔK/ΔL| = MPL / MPK**
- MRTS **diminishes** as you move along an isoquant (law of diminishing MRTS)

#### Mini Example:
> Producing 10 pairs of shoes:  
> Points: a(K=8, L=2), b(K=4, L=4), c(K=2, L=8)  
> MRTS (a→b) = |4−8|/|4−2| = **4/2 = 2**  
> MRTS (b→c) = |2−4|/|8−4| = **2/4 = 0.5** ← diminishing

---

### 7.11 Three Types of Isoquant Shapes

```
Panel I  → Right angles    = FIXED proportion (no substitution, MRTS = 0)
           e.g., tyres + battery in a car

Panel II → Straight lines  = PERFECT substitutes (constant MRTS)
           e.g., natural gas vs fuel oil

Panel III→ Curved (convex) = IMPERFECT substitutes (most common) ✓
```

---

### 7.12 Economic Region of Production
- Ridge lines OA and OB bound the **economic region**
- Outside ridge lines: MP of one input is negative → economically inefficient
- Rational firms produce **within** the ridge lines only

---

### 7.13 Isocost Lines

- **Isocost:** All combos of K and L a firm can buy for a given budget C
- Formula: **C = PL × L + PK × K**
- Slope of isocost = **−PL/PK**
- If labour price rises → isocost rotates inward on the L-axis

#### Mini Example:
> PL = 10, PK = 20, Budget = 200  
> Max K = 200/20 = **10 units**  
> Max L = 200/10 = **20 units**  
> Slope = −10/20 = **−0.5**

---

### 7.14 Optimal Input Combination (Long Run)

- Optimal point = where **isoquant is TANGENT to isocost**
- Condition: **MPL/PL = MPK/PK** (equal output per rupee spent on each input)
- Equivalently: **MPL/MPK = PL/PK**

```
Too much K (Point A) → MPL/PL > MPK/PK → substitute L for K
Too much L (Point B) → MPL/PL < MPK/PK → substitute K for L
Optimal   (Point Q) → MPL/PL = MPK/PK ✓
```

---

### 7.15 Returns to Scale

```
All inputs increase by 10%:

Output increases > 10%  → INCREASING Returns to Scale (IRS)
Output increases = 10%  → CONSTANT Returns to Scale (CRS)
Output increases < 10%  → DECREASING Returns to Scale (DRS)
```

- Public utilities (telecom, electricity) often show IRS
- Whether an industry shows CRS/IRS/DRS is an **empirical question**

---

### 7.16 Cobb-Douglas Production Function

- **Formula: Q = A · Kᵅ · Lᵝ**
- A, α, β are constants estimated by regression
- Returns to scale determined by **(α + β)**:
  - α + β > 1 → IRS
  - α + β = 1 → CRS
  - α + β < 1 → DRS
- Linearised by taking logs: **log Q = log A + α log K + β log L**

---

### 7.17 Types of Statistical Analysis for Estimating Production Functions

```
TIME SERIES    → Same firm over time (assumes constant technology)
CROSS-SECTION  → Many firms at one point in time
ENGINEERING    → Technical/experimental data (not limited to past observations)
```

---

### 7.18 Managerial Uses of Production Function
- Find **least-cost input combination** for a target output
- Decide how much of a variable input to add (use until MRP = input price)
- Guide **long-run expansion** decisions based on returns to scale

## Chapter 8: Short Run Cost Analysis
**Overview:** Costs are classified in many ways for different decisions. This chapter defines key cost concepts and shows how short-run cost curves behave and how to apply them in break-even and profit analysis.

---

### 8.1 Key Cost Concepts

#### Actual vs Opportunity Cost
- **Actual (Accounting) Cost:** Money actually paid out (wages, rent, materials)
- **Opportunity Cost:** Value of the next best alternative foregone
  - Example: Owner runs own shop → opportunity cost = salary foregone from a regular job

#### Explicit vs Implicit Cost
- **Explicit:** Cash payments to outsiders (wages, rent, advertising)
- **Implicit:** Value of owner-owned resources used (no cash changes hands)
  - Example: Owner's unpaid salary, interest on own capital

#### Accounting vs Economic Profit

```
Accounting Profit = Revenue − Explicit Costs
Economic Profit   = Revenue − (Explicit + Implicit Costs)
```

#### Mini Example (Mr. Raj's Store):
| Item | Accountant | Economist |
|---|---|---|
| Sales | ₹8,00,000 | ₹8,00,000 |
| Explicit Costs | ₹6,50,000 | ₹6,50,000 |
| Implicit Costs | — | ₹75,000 |
| **Profit** | **₹1,50,000** | **₹75,000** |

---

### 8.2 More Cost Pairs

```
FIXED COSTS    → Don't change with output (rent, depreciation)
VARIABLE COSTS → Change with output (wages, raw materials)
SEMI-VARIABLE  → Partly fixed, partly variable (e.g., depreciation)

DIRECT COSTS   → Traced to a specific product (raw material, labour)
INDIRECT COSTS → Shared/overhead (electricity, admin expenses)

SUNK COSTS     → Already spent, cannot be recovered → IRRELEVANT for decisions
INCREMENTAL    → Extra costs from a decision → RELEVANT for decisions

PAST COSTS     → Historical, recorded in books
FUTURE COSTS   → Estimated; used for managerial decisions ✓

PRIVATE COSTS  → Borne by firm
SOCIAL COSTS   → Also borne by society (e.g., pollution dumped in river)
```

---

### 8.3 Short Run Cost Functions

#### Three Total Cost Concepts:
- **TFC** = Total Fixed Cost (constant, e.g., ₹100 regardless of output)
- **TVC** = Total Variable Cost (rises with output)
- **TC** = TFC + TVC

```
TC
|          /TC
|         /
|        /TVC
|_______/
|_______TFC (flat line)
|________________________ Output (Q)
```

---

### 8.4 Average & Marginal Costs

- **AFC** = TFC / Q → always falls as Q increases (spreads fixed cost)
- **AVC** = TVC / Q → U-shaped
- **ATC** = TC / Q = AFC + AVC → U-shaped, always above AVC
- **MC** = ΔTC / ΔQ = ΔTVC / ΔQ → U-shaped, rises due to diminishing returns

#### Mini Example (from Table 8.1):
> At Q = 5: TFC = 100, TVC = 150, TC = 250  
> AFC = 100/5 = **₹20**, AVC = 150/5 = **₹30**, ATC = 250/5 = **₹50**  
> MC = (250 − 240) = **₹10** (change from Q=4 to Q=5)

---

### 8.5 Relationships Between Cost Curves

```
MC cuts both AVC and ATC at their MINIMUM points (from below)

MC < AVC → AVC is falling
MC = AVC → AVC is at minimum ✓
MC > AVC → AVC is rising

Same logic applies for ATC
```

---

### 8.6 Link Between Productivity and Cost

- **AVC = W / AP** → AVC is minimum when AP is maximum
- **MC = W / MP** → MC is minimum when MP is maximum
- When MP rises → MC falls; when MP falls → MC rises (mirror image)

---

### 8.7 Applications of Short Run Cost Analysis

#### A. Optimal Output Level
> Find Q where **AC = MC** (minimum average cost)

**Mini Example:**
> TC = 128 + 6Q + 2Q²  
> AC = 128/Q + 6 + 2Q  
> Set AC = MC: 128/Q + 6 + 2Q = 6 + 4Q  
> → 128/Q = 2Q → Q² = 64 → **Q = 8**

---

#### B. Break-Even Analysis

```
BREAK-EVEN POINT = Output where Total Revenue = Total Cost (zero profit)

Formula: Q_BE = TFC / (P − AVC)
```

**Mini Example:**
> TFC = ₹500, Price = ₹5, AVC = ₹4  
> Q_BE = 500 / (5 − 4) = **500 units**  
> Break-even Revenue = 500 × ₹5 = **₹2,500**

```
Revenue / Cost
    |              /TR
    |             /
    |   Profit→ /
BE→ |----------/
    |  Loss  / /TC
    |       / /
    |______/__________
         500      Q
```

---

#### C. Profit Contribution Analysis

- **Profit Contribution per unit = Price − AVC**
- Each unit sold first covers fixed costs, then generates profit
- Formula to hit a target profit (π_R):

```
Q = (π_R + TFC) / (P − AVC)
```

**Mini Example:**
> Target profit = ₹10,000; TFC = ₹500; P = ₹5; AVC = ₹4  
> Q = (10,000 + 500) / (5 − 4) = **10,500 units**

---

#### D. Operating Leverage (E_π)

- Measures how sensitive **profit** is to a change in **sales volume**
- Higher fixed costs → higher operating leverage → bigger profit swings

```
Formula: E_π = Q(P − AVC) / [Q(P − AVC) − TFC]
```

**Mini Example (3 firms, Q = 40,000):**

| Firm | Fixed Cost | AVC | Price | E_π |
|------|-----------|-----|-------|-----|
| I    | ₹1,00,000 | ₹2  | ₹5   | **6** |
| II   | ₹60,000   | ₹3  | ₹5   | **4** |
| III  | ₹26,650   | ₹4  | ₹5   | **3** |

> Firm I: 1% rise in sales → **6% rise in profit** (most sensitive)

---

### 8.8 Summary Table — When to Use Which Cost Concept

| Decision | Relevant Cost |
|---|---|
| Break-even output | TC, TFC, AVC |
| Expand production or not | MC vs MR |
| Per-unit profit | ATC |
| Short-term pricing | AVC (floor price) |
| Choosing production method | Incremental / Opportunity cost |
| Past investment write-off | Sunk cost (ignore it) |
